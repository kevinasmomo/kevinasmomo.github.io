Something new

Something new

当我告诉别人我在做自己的博客的时候，很多人都问我：

——新浪？

——no！我打算自己做一个独立的博客。

——你会吗？

——我用Hexo和Github，很方便。

——那是什么，我不知道，你到时候来教教我呗。

所以，什么是Hexo，Github？

Hexo就是一个简单地、轻量地、基于Node的一个静态博客框架。

A fast, simple & powerful blog framework, powered by Node.js.

Features

  • Blazing fast generating
  • Support for GitHub Flavored Markdown and most Octopress plugins
  • One-command deploy to GitHub Pages, Heroku, etc.
  • Powerful plugin system

简而言之，快速、简单且功能强大的 Node.js 博客框架。

Github

Git 是由 Linux 之父 Linus Tovalds 为了更好地管理linux内核开发而创立的分布式版本控制／软件配置管理软件。

当然对于我来说其本身提供免费的托管服务，又提供了贴心的 Pages 功能，可以绑定自己的域名，免费、高效、不限流量，做一个个人页面绰绰有余。（没钱只能折腾）

不过嘛，如果你们也想用这些来搭建自己的博客的话，请自己百度。我相信自己可以靠百度解决大多数问题，你们也一定可以。

所以在这篇里，我是不会介绍具体的操作，喜欢什么东西，自己去做就好了。

Markdown

Markdown真的是一个好用的东西，比起以前在公众号上的编辑体验，更加轻松、快捷、简单，所以这篇博客的所有文章都是基于Markdown编辑的。

我在这里也只是简单的介绍了一下这些我觉得很新鲜的东西，我也知道自己掌握的都是一些很皮毛的东西。而且相对于其他博客平台来说，的确很难上手，很多问题都要自己一点点找到根源去解决。所以我在这里也只是放上了我做这个博客用到的东西，具体的操作，还是自己去寻找吧。

如果一开始知道难就不去做，那才是真的可惜了。

Have a nice day!
